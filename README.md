# Custom Junie Skills

This repository contains my personal custom skills for [Junie](https://www.jetbrains.com/), the autonomous programmer by JetBrains.

## Overview

Junie skills extend the assistant's capabilities with specialized instructions, domain-specific workflows, and resources. 

> **Note:** Since skills are updated frequently (with new skills added or existing ones modified over time), this README does not list or document every individual skill. You can explore the repository structure to see the available skills.

## Installation

To install these custom skills, follow these step-by-step instructions:

- **Step 1:** Open your terminal (UNIX/macOS) or command prompt/PowerShell (Windows).
- **Step 2:** Navigate to your Junie skills directory:
  - **UNIX / macOS:** `cd ~/.junie/skills`
  - **Windows:** `cd %HOMEPATH%/.junie/skills`
- **Step 3:** Clone the repository directly into the root of the skills directory using a dot `.` at the end so the skill folders reside directly at the root level:
  - `git clone <repository-url> .`

> **Important:** Cloning directly into the root folder ensures that Junie can correctly read and recognize all custom skills.
